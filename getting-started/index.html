Getting Started

Step 1: Log In To Soluble Fusion

This is fast and simple, using OpenID Connect.

  • Open a browser to https://app.soluble.cloud

  • Click Create Account

  • Select Google or GitHub indentity providers (let us know if you'd like to see other identity providers)
  • Select your account
  • Agree to Terms of Service
  • Done!

Step 2: Install the CLI

Follow the instruction to install the CLI in the sidebar.

There are more detailed insructions available in the documentation.

Step 3: Run your first assessment

Enter a directory containing terraform or cloudformation and run the following:

$ soluble iac-scan

This will show some of the assessment tools that are available.

# run checkov:
soluble iac-scan checkov

# run tfsec:
soluble iac-scan tfsec

# run cfn-lint:
soluble iac-scan cfn-python-lint

To send the results to Soluble Fusion, add the --upload flag. For instance:

$ soluble iac-scan checkov --upload
...

The output will include a link to the assessment results.

Notes

For tools that depend on a language runtime and installation (i.e. checkov, cfn-lint need python runtime) we use a containerized distribution by default.

If you don't have docker available or prefer to use the already-installed version of the assesment tool, you can add --tool-path <executable>.

So to use the locally installed version of cfn-lint:

# To use cfn-lint on your PATH:

soluble iac-scan cfn-lint --tool-path cfn-lint

# Sepecify a specific installation:

soluble iac-scan cfn-lint --tool-path /usr/local/bin/cfn-lint
